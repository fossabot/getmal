# Get MyAnimeList! `(*°▽°*)`
Returns the data of a particular anime list and status in JSON.

### Using
`https://getmal.herokuapp.com/LIST/USERNAME/STATUS`

### Lists:
```
anime - for anime list
manga - for manga list
```

### Status codes:
```
7: All Anime
1: Currently Watching
2: Completed
3: On Hold
4: Dropped
6: Plan to Watch
```
